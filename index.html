Bomberman / DynaBlaster

We enjoyed many hours playing Dynablaster in the mid-90' so we decided to create a death match for us. The last update was 0.6, what we called 0.6nobugs ;) in 12-Jan-2000 and was hosted at brinkster. It was developed for a 320x200 resolution mode13 as our first PC was a PC1512.

  • Game technique. http://www.dusk-and-dawn.com/?p=132 http://www.kenney.nl/assets/platformer-pack-redux
  • How did we created it. At that time there were several compilers that we were aware of: DJGPP Allegro/DJGCC, Borland Turbo-C and Watcom. We had Turbo-C as our key dev environment, and use some libraries to managed graphics mode (implemented with asembler) You may find several other compilers at wikipedia compiler list or compilers.net site. It didn't have music or sounds, as we focus on silent matches ;)
  • How to run it. The best place to run this game would be a DOS x86 environment such as FreeDos, DosBox, Boschs, DosEMU, Fake86. We were using several OS, such as DR-DOS, MS-DOS, Caldera OpenDOS
